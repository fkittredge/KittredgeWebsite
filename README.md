# Kittredge Technologies Website

Professional consulting website for Fletcher Kittredge and Kittredge Technologies.

## Features
- Responsive design with Harvard Crimson and Hunter Green color scheme
- Professional timeline showcasing 30+ years of technology leadership
- Career highlights from BBN Internet pioneers to GWI transformations
- Services: Technology transformation consulting, grant writing, executive advisory
- Contact information and professional background

## Technology Stack
- HTML5 with semantic markup
- CSS3 with responsive design
- Vanilla JavaScript (minimal)
- No build process required

## Local Development

### Prerequisites
- Python 3.x (for local server)

### Running Locally
```bash
# Start local development server
python -m http.server 8000

# View at http://localhost:8000
```

## Deployment

### Netlify (Recommended)
This site is configured for automatic deployment on Netlify:

1. Connect your GitHub repository to Netlify
2. Build settings are configured in `netlify.toml`
3. Automatic deployments on every push to main/master branch

### Manual Deployment
Upload the following files to any web server:
- `index.html`
- `styles.css`
- `CLAUDE.md` (optional, for documentation)

## Project Structure
```
├── index.html          # Main website page
├── styles.css          # All styling and responsive design
├── netlify.toml        # Netlify deployment configuration
```
